# 🤖 Interactive AI News Intelligence Bot

A live, interactive Telegram bot that acts as a personal research assistant, fetching real-time global news based on user-defined topics.

## 🚀 Live Demo
You can try the bot right now on Telegram: [@YourBotUsername]

## ✨ Features
- **On-Demand Search:** Use `/news <topic>` to get the latest 3 articles on any subject.
- **24/7 Availability:** Hosted on Render with a Flask-based "Keep-Alive" system.
- **Real-Time Data:** Powered by NewsAPI for the most recent global headlines.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **APIs:** Telegram Bot API (via pyTelegramBotAPI), NewsAPI
- **Backend:** Flask (for web server threading)
- **Deployment:** Render (Cloud Hosting)
